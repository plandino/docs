## Build Mini Apps for Lemon Cash

Create powerful mini applications that run inside the Lemon Cash mobile app using React Native WebView technology. Our SDK provides everything you need to build secure, interactive mini apps that can authenticate users, handle transactions, and interact with smart contracts.
