## Build your first mini app

Create a simple mini app that authenticates users and handles deposits in just a few steps.

### Step 1: Install and set up

<AccordionGroup>
  <Accordion icon="download" title="Install the SDK">
    Install the Mini App SDK in your project:
    
    ```bash
    npm install mini-app-sdk
    ```
    
    <Tip>You can also use yarn or pnpm as your package manager.</Tip>
  </Accordion>
  <Accordion icon="code" title="Create your mini app">
    Create a new React component for your mini app:
    
    ```typescript
    import React from 'react';
    import { useAuthenticate, DepositButton, useWebView } from 'mini-app-sdk';
    
    function MyMiniApp() {
      const isWebView = useWebView();
      
      if (!isWebView) {
        return <div>This app only works inside Lemon Cash</div>;
      }
      
      return <div>Welcome to your mini app!</div>;
    }
    
    export default MyMiniApp;
    ```
  </Accordion>
</AccordionGroup>

### Step 2: Add authentication

<AccordionGroup>
  <Accordion icon="key" title="Implement user authentication">
    Add authentication to your mini app using the `useAuthenticate` hook:
    
    ```typescript
    function MyMiniApp() {
      const { authenticate, loading, error, data } = useAuthenticate({
        nonce: 'your-unique-nonce',
        onSuccess: (response) => {
          console.log('Authenticated:', response.predictedSafeAddress);
        },
        onError: (error) => {
          console.error('Authentication failed:', error);
        },
      });
      
      return (
        <div>
          <button onClick={authenticate} disabled={loading}>
            {loading ? 'Authenticating...' : 'Authenticate'}
          </button>
          {error && <p>Error: {error.message}</p>}
          {data && <p>Address: {data.predictedSafeAddress}</p>}
        </div>
      );
    }
    ```
    
    <Tip>The authentication process uses wallet signatures for secure user verification.</Tip>
  </Accordion>
  <Accordion icon="wallet" title="Add deposit functionality">
    Include a deposit button in your mini app:
    
    ```typescript
    function MyMiniApp() {
      return (
        <div>
          <DepositButton
            amount={100}
            currency="USDC"
            network="polygon"
            text="Deposit 100 USDC"
            onClick={() => console.log('Deposit initiated')}
          />
        </div>
      );
    }
    ```
  </Accordion>
</AccordionGroup>

### Step 3: Test and deploy

<Accordion icon="rocket" title="Test your mini app">
  1. Deploy your mini app to a web server.
  2. Test it inside the Lemon Cash mobile app.
  3. Verify that authentication and deposits work correctly.
  
  <Warning>
  The SDK features only work when running inside the Lemon Cash WebView environment.
  </Warning>
</Accordion>

## Complete example

Here's a complete mini app that combines authentication and deposit functionality:

```typescript
import React, { useState } from 'react';
import { 
  useAuthenticate, 
  DepositButton, 
  useWebView,
  useDepositResponse 
} from 'mini-app-sdk';

function MyMiniApp() {
  const isWebView = useWebView();
  const [isAuthenticated, setIsAuthenticated] = useState(false);
  
  const { authenticate, loading: authLoading, error: authError, data: authData } = useAuthenticate({
    nonce: 'your-unique-nonce-' + Date.now(),
    onSuccess: (response) => {
      setIsAuthenticated(true);
      console.log('Authenticated:', response.predictedSafeAddress);
    },
    onError: (error) => {
      console.error('Authentication failed:', error);
    },
  });
  
  const { txHash, loading: depositLoading } = useDepositResponse({
    onSuccess: () => {
      console.log('Deposit successful:', txHash);
    },
    onError: () => {
      console.error('Deposit failed');
    },
  });
  
  if (!isWebView) {
    return <div>This app only works inside Lemon Cash</div>;
  }
  
  return (
    <div>
      <h1>My Mini App</h1>
      
      {!isAuthenticated ? (
        <div>
          <button onClick={authenticate} disabled={authLoading}>
            {authLoading ? 'Authenticating...' : 'Authenticate'}
          </button>
          {authError && <p>Error: {authError.message}</p>}
        </div>
      ) : (
        <div>
          <p>Welcome! Address: {authData?.predictedSafeAddress}</p>
          <DepositButton
            amount={100}
            currency="USDC"
            network="polygon"
            text="Deposit 100 USDC"
            loading={depositLoading}
          />
          {txHash && <p>Transaction: {txHash}</p>}
        </div>
      )}
    </div>
  );
}

export default MyMiniApp;
```
